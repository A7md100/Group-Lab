This project aims to develop a simple web app for the Rock, Paper, Scissors game that can be sold to other companies for team-building purposes.
The app will allow two players to play the game and keep track of the score for any rounds. The scope of the project includes the following:
- Creating a user interface that is intuitive and easy to use
- Developing the game logic to ensure that it follows the rules of Rock, Paper, Scissors
- Writing code that is well-organized, efficient, and easy to maintain
- Testing the code thoroughly to ensure that it functions correctly and meets the quality standards
- Coordinating releases and updates as needed to ensure timely delivery to customers
